## рџљЂ Getting Started

### Prerequisites
- Node.js (v14.0.0 or higher)
- npm or yarn
- Git

### Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/ashishxa/portfolio.git
cd portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm start
```
The app will open at [http://localhost:3000](http://localhost:3000)

4. **Build for production**
```bash
npm run build
```

## вњЏпёЏ How to Customize

### Update Personal Information

Navigate to `/src/components/` and edit the following files:

1. **About Section** - `About/AboutCard.js`
   - Update name, designation, university, and experience
   
2. **Home Section** - `Home/Home2.js`
   - Modify introduction text and key interests

3. **Projects** - `Projects/Projects.js`
   - Add/remove/update project cards
   - Update project links and descriptions

4. **Skills** - `About/Techstack.js`, `About/Toolstack.js`
   - Update technical skills and tools

5. **Resume** - `Resume/ResumeNew.js`
   - Link your resume file

### Update Social Media Links

Edit these files to update social profiles:

- `components/Home/Home.js`
- `components/Footer.js`

Current social links:
- GitHub: https://github.com/ashishxa
- X (Twitter): https://x.com/ashishkumarr__
- Instagram: https://www.instagram.com/pvt.aishx/

### Customize Colors

Edit `src/style.css` to change the theme colors:

```css
/* Change the primary purple color */
--section-title-color: #purple-color;
```
